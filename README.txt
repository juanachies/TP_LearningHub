LearningHub:
Se trata de una pagina web pensada por y para estudiantes. Se diseñó con la idea de facilitar el acceso a información y recursos descargables.

- Se puede ingresar a la página a través de este link: 

- O puede correr el proyecto localmente siguiendo estos pasos:
  1. Podes activar el entorno virtual con 'entorno_virtual/Scripts/activate', o crear uno nuevo con 'python -m venv entorno_virtual' y 'pip install flask flask_sqlalchemy' (para descargar lo que necesitamos)
  2. Ingresar 'python app.py' en la terminal
  3. Ingresar a 'http://127.0.0.1:8000/' desde Google
